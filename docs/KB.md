## 조회 조건 요약

| 파라미터 | 값 | 설명 |
|---|---|---|
| `지역코드` | 시도 2자리 (11=서울, 26=부산, 41=경기...) | 넣으면 시군구까지, 안 넣으면 시도만 |
| `기간` | '1'~'10' 또는 미지정=전체 | 최근 N년 |
| `매물종별구분` | 01=아파트, 08=연립, 09=단독, 98=종합 | |
| `매매전세코드` | 01=매매, 02=전세 | |
| `월간주간구분코드` | 01=월간, 02=주간 | |
| `메뉴코드` | 01~06 (시장동향용) | |

## 1회 호출 = 1시도의 전체 시군구 × 요청기간

예: 지역코드='11', 기간='10' → 서울 26개 구 × 120개월 = 약 3,120행 한번에 옴

전국 시군구를 다 받으려면 시도 17개 × 호출 = **17번 호출**이면 끝.

결과를 다시 보면:

주간 가격지수 / 주간 증감률 → 월간주간구분코드: '02'로 호출하면 주간 단위
나머지는 전부 월간
정리하면:

주기	해당 지표
주간	매매/전세 가격지수, 매매/전세 가격지수 증감률, 매수우위지수, 매매거래활발지수, 전세수급지수, 전세거래활발지수
월간	위 지표 전부 + ㎡당 평균가격, 평균가격, 중위가격, 전세가격비율, 전월세전환율, 매매가격전망지수, 전세가격전망지수, 면적별 가격지수, PIR, 월세가격지수
주간 데이터가 있는 지표는 주간/월간 둘 다 제공합니다. 파라미터로 선택하는 구조입니다.

회사 제출용 10년치는 월간으로 충분하고, 공인중개사 대응용은 최신 동향을 보여주려면 주간이 유리합니다.
